wiCAN help page

Aplicația wiCAN este o aplicație web bazata pe javascript care permite controlul și recepționarea mesajelor de pe dispozitivul wiCAN. In continuare se va prezenta principiile de baza in lucrul cu aceasta aplicație.

Crearea și logarea utilizatorilor

Logarea utilizatorilor se executa folosind un e-mail și parolă specificate la înregistrare. Astfel pentru a ne autentifica trebuie să introducem datele în zonele necesare și să apăsăm butonul ”Login”.

În cazul în care suntem un utilizator nou și avem nevoie de un cont nou este necesar să apasăm butonul ”Create new user” și va apare un dialog unde este necesar sa introduceți datele personale cum ar fi Numele, e-mail și parola. După introducerea datelor apăsați butonul ”Create an account”.

Descrierea spațiului de lucru

Spațiul de lucru se împarte în mai multe zone:

  1. Zona utilizatorului
  2. Zona de control a proiectelor
  3. Zona de afișare a proiectelor
  4. Zona de lucru cu proiectul
1. În zona utilizatorul sunt disponibile două butoane: ”Logout” care servește pentru delogarea din sistem și ”Help” pentru afișarea acestei pagini.
2. În zona de control a proiectelor sunt disponibile butoane pentru modificarea și controlul proiectelor. Butoanele inițial sunt dezactivate și se activeaza doar în cazul când utilizatorul selectează anumite zone ale proiectului. Lista butoanelor și destinația lor este prezentată mai jos:
  • Create project - Crearea unui nou proiect.
  • Remove project - Ștergerea unui proiect din listă.
  • Open project - Deschiderea proiectului selectat
  • Run project - Rularea proiectului. Prusupune transmiterea, recepționarea și afșarea mesajelor inițial configurate.
  • Create workspace - Crearea unui nou spațiu de lucru pentru controlul sau afișarea mesajelor.
  • Remove workspace - Ștergerea spațiului de lucru selectat.
  • Create ECU - Crearea unui nou ECU (Electronic Control Unit).
  • Remove ECU - Ștergerea unui ECU selectat.
  • Create message - Crearea unui mesaj pentru transmisie sau recepție, în funție de folderul selectat.
  • Remove message - Ștergerea mesajului selectat.
  • Edit message - Editarea mesajului selectat.
  • Upload DBC file - Încaărcarea unui fișier de format DBC, care descrie structura proiectului în zona de mesaje.
  • Create view - Crearea unei vizualizari a unui mesaj în spațiul de lucru selectat.
  • Create control - Crearea unui nou control al unui mesaj în spațiul de lucru selectat.
3. În această zonă este afișată lista de proiecte disponibile și în cazul unui proiect deschis, acestuia i se fac disponibile subdomeniile respective.
4. Zona aceasta este destinată controlului și vizualizării rezultatelor de transmitere și recepționare a mesajelor pentru fiecare spațiu de lucru cu mesajele create anticipat.
